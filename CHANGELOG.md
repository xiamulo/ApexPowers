# Changelog

## 0.1.0

- Added `apex` CLI for profile install, update, uninstall, doctor, agent mirror sync, hooks, and packing.
- Added user-facing shortcut commands: `init`, `add`, `remove`, and `sync`.
- Added profile registry with `core`, `hooks`, `planning`, `research`, `frontend`, `quality`, `gsap`, and `full`.
- Added Codex and Claude Code plugin manifests.
- Added distribution checker, SBOM-lite artifact metadata, SHA256SUMS generation, and benchmark script.
- Added open-source project metadata and release scaffolding.
